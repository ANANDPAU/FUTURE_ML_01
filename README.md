## 📌 Project Overview

This project focuses on forecasting future sales demand using historical retail business data. The objective is to apply Machine Learning and Time Series Analysis techniques to predict sales trends and generate business insights.

This project demonstrates a complete ML workflow including data preprocessing, visualization, model building, evaluation, and forecasting.

---

## 🎯 Objectives

* Perform sales trend analysis
* Build a forecasting model
* Predict future sales demand
* Visualize business trends
* Generate actionable business insights

---

## 🗂 Dataset Information

Dataset contains retail sales transaction data.

**Total Records:** 9994
**Total Features:** 21

### Important Features:

* Order Date (Time Feature)
* Sales (Target Variable)
* Profit
* Quantity
* Discount
* Category
* Sub-Category
* Region
